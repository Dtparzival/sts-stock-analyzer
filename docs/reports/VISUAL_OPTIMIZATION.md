# 美股投資分析平台 - 視覺優化說明文件

## 優化概述

本次優化針對美股投資分析平台進行全方位的視覺設計和使用者體驗改進，包括數字顯示專業化、金色點綴增強、動畫效果、字體層級、色彩對比度等多個方面，確保跨裝置使用體驗一致。

## 優化內容

### 1. 全域樣式系統優化

#### 1.1 字體系統
- **引入 IBM Plex Mono 等寬字體**：專門用於財務數據和數字顯示，提升專業感和可讀性
- **字體來源**：Google Fonts CDN
- **應用範圍**：所有價格、百分比、財務指標數字

#### 1.2 金色點綴效果
新增金色漸層 CSS 變數和工具類別：

```css
/* 金色漸層效果（用於 CTA 按鈕和重要指標） */
--gradient-gold: linear-gradient(135deg, oklch(0.80 0.15 85) 0%, oklch(0.70 0.18 75) 50%, oklch(0.75 0.15 80) 100%);
--gradient-gold-hover: linear-gradient(135deg, oklch(0.85 0.18 85) 0%, oklch(0.75 0.20 75) 50%, oklch(0.80 0.18 80) 100%);
```

**工具類別**：
- `.bg-gradient-gold` - 金色漸層背景
- `.border-gold` - 金色邊框
- `.text-gold` - 金色文字
- `.shadow-gold` / `.shadow-gold-lg` - 金色陰影效果

#### 1.3 進場動畫系統
新增多種進場動畫效果：

- **`.animate-fade-in`** - 淡入動畫（0.5s）
- **`.animate-slide-up`** - 向上滑入動畫（0.6s）
- **`.animate-slide-in-left`** - 從左滑入動畫（0.6s）
- **`.animate-slide-in-right`** - 從右滑入動畫（0.6s）
- **`.animate-delay-100/200/300/400`** - 延遲動畫（用於錯開顯示）

#### 1.4 載入動畫優化
- **`.animate-pulse-blue`** - 藍色脈衝效果（使用專業信任藍配色）
- **`.animate-spin-slow`** - 慢速旋轉動畫（3s）

#### 1.5 陰影效果增強
新增藍色和金色陰影效果：

- **藍色陰影**：`.shadow-blue` / `.shadow-blue-lg`
- **金色陰影**：`.shadow-gold` / `.shadow-gold-lg`

### 2. 首頁視覺優化

#### 2.1 標題動畫效果
- 主標題添加 `.animate-slide-up` 向上滑入動畫
- 副標題添加 `.animate-fade-in` 淡入動畫（延遲 100ms）

#### 2.2 CTA 按鈕優化
- 搜尋按鈕改用金色漸層背景（`.bg-gradient-gold`）
- 添加金色陰影效果（`.shadow-gold-lg`）
- 優化 hover 狀態動畫

#### 2.3 功能卡片優化
- **即時股票數據卡片**：藍色陰影效果（`.shadow-blue`）
- **AI 投資分析卡片**：金色漸層背景 + 金色陰影（`.bg-gradient-gold` + `.shadow-gold`）
- **投資組合管理卡片**：保持原有藍色漸層設計
- 所有卡片添加錯開淡入動畫（延遲 100ms/200ms/300ms）

#### 2.4 推薦卡片數字顯示
- 股價和漲跌幅使用等寬字體（`.number-display`）
- 確保數字對齊整齊，提升專業感

### 3. 股票詳情頁優化

#### 3.1 價格資訊顯示
所有財務數據使用等寬字體（`.number-display`）：
- 當前價格
- 漲跌金額和漲跌幅
- 開盤價、昨收價、最高價、最低價
- 市值

#### 3.2 歷史分析記錄
- 分析記錄中的股價使用等寬字體
- 優化數字對齊和可讀性

### 4. 我的收藏頁面優化

#### 4.1 股價顯示組件
- 當前價格使用等寬字體（`.number-display`）
- 漲跌幅數字使用等寬字體
- 確保與首頁推薦卡片風格一致

### 5. 響應式設計優化

#### 5.1 已實現的響應式優化
- 所有動畫效果在小螢幕上自動優化性能
- 漸層效果在移動裝置上減少複雜度
- 觸控友好的按鈕尺寸（最小 44x44px）

#### 5.2 響應式工具類別
系統已包含完整的響應式工具類別：
- 響應式文字大小（`.text-responsive-*`）
- 響應式間距（`.gap-responsive`、`.p-responsive`）

- 響應式按鈕尺寸（`.btn-responsive`）

## 使用指南

### 如何應用金色漸層效果
```tsx
// 按鈕
<Button className="bg-gradient-gold hover:bg-gradient-gold-hover">
  立即開始
</Button>

// 卡片
<Card className="shadow-gold border-gold">
  重要內容
</Card>
```

### 如何應用數字顯示樣式
```tsx
// 價格顯示
<div className="number-display text-2xl font-bold">
  ${price.toFixed(2)}
</div>

// 百分比顯示
<div className="number-display text-lg font-semibold">
  {changePercent.toFixed(2)}%
</div>
```

### 如何應用進場動畫
```tsx
// 標題動畫
<h1 className="animate-slide-up">
  智慧投資
</h1>

// 卡片錯開動畫
<Card className="animate-fade-in animate-delay-100">
  內容 1
</Card>
<Card className="animate-fade-in animate-delay-200">
  內容 2
</Card>
```

## 維護注意事項

### 保持一致性
- 所有新增的財務數據顯示都應使用 `.number-display` 類別
- 重要 CTA 按鈕優先使用金色漸層效果
- 功能卡片根據重要性選擇藍色或金色陰影

### 性能考量
- 避免在同一頁面使用過多動畫效果
- 大型列表應使用虛擬滾動優化性能
- 移動裝置上自動減少漸層和陰影複雜度

### 可訪問性
- 確保所有動畫都可以通過 CSS `prefers-reduced-motion` 禁用
- 保持足夠的色彩對比度（至少 4.5:1）
- 確保所有互動元素有足夠的觸控區域（最小 44x44px）
